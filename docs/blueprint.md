# **App Name**: DormFix

## Core Features:

- Request Submission: Allow students to submit maintenance requests with details: room number, category (AC, plumbing, etc.), priority, description, optional photo.
- Request Listing: Display a list of submitted requests with key information: room number, category, priority, status, and created date.
- Request Filtering: Enable filtering and sorting of maintenance requests based on category, priority, and status.
- Report Generation: Download a report of all maintenance requests in a docx format.
- AI Powered Suggestions: Use an AI tool to prioritize requests, suggest optimal maintenance schedules, and flag duplicate requests.
- Intuitive UI: Implement a user-friendly interface with clear instructions and easy navigation.
- Duplicate Detection: Analyze all hostel maintenance requests and automatically detect duplicates if they come from the same room.
- Urgency Prediction: Predict urgency for safety-critical issues, such as lift malfunctions trapping a person or electrical hazards, and flag them for immediate attention.
- Daily Summaries Dashboard: Generate concise daily summaries highlighting total requests, most common issues, and any urgent clusters in a dashboard

## Style Guidelines:

- Primary color: A calming blue (#64B5F6), promoting trust and efficiency in maintenance processes.
- Background color: A very light blue-gray (#F0F4F8), which gives a clean and professional feel.
- Accent color: A contrasting orange (#FF8A65) to highlight important actions like submitting a request.
- Font: 'Inter' sans-serif for both headlines and body text, chosen for its modern and readable design. Inter is suitable for both headlines and body text.
- Use simple, clear icons to represent different maintenance categories (e.g., wrench for plumbing, lightbulb for electrical).
- Prioritize a clean and organized layout for easy navigation, with clear sections for submitting, viewing, and filtering requests.
- Incorporate subtle animations for loading states and form submissions to enhance the user experience.
